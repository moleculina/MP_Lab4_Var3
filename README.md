# 🎓 University Portal - Система управления успеваемостью студентов
# 📝 Описание проекта
Учебный web-проект по дисциплине «Межплатформенное программирование», демонстрирующий работу с реляционной базой данных SQLite, контейнеризацию приложений и взаимодействие Fullstack-частей (Backend + Frontend).

Вариант лабораторной работы: №3 - Университет

Предметная область: Система управления успеваемостью студентов. Позволяет добавлять студентов и дисциплины, выставлять оценки, просматривать рейтинг студентов, средний балл и список отличников.

## Структура проекта

```text
university_portal/
├── 📁 backend/                    # Серверная часть приложения
│   ├── 📁 app/                    # Исходный код API и логика БД
│   │   ├── main.py                # Точка входа и инициализация FastAPI
│   │   ├── database.py            # Настройка подключения к SQLite
│   │   ├── models.py              # ORM-модели таблиц (Students, Disciplines, Grades)
│   │   ├── schemas.py             # Pydantic-схемы для валидации данных
│   │   └── routes.py              # Маршрутизация: обработка путей и запросов (эндпоинтов)
│   └── Dockerfile                 # Инструкции сборки образа бэкенда
├── 📁 frontend/                   # Клиентская часть (статические файлы)
│   ├── index.html                 # Главная страница интерфейса
│   ├── style.css                  # Стили оформления (с поддержкой тёмной/светлой темы)
│   ├── script.js                  # Логика запросов к API
│   ├── nginx.conf                 # Конфигурация веб-сервера Nginx
│   └── Dockerfile                 # Инструкции сборки образа фронтенда
├── 📁 data/                       # Папка для файла базы данных (создаётся автоматически)
│   └── university.bd              #Файл базы данных SQLite   
├── .gitignore                     #Файл с перечнем игнорируемых файлов для GIT
├── docker-compose.yml             # Файл оркестрации всех сервисов
├── requirements.txt               # Зависимости Python
└── README.md                      # Документация проекта
```
# 🎯 Назначение проекта
## Данная работа демонстрирует ключевые навыки разработки:
* Проектирование БД: Работа с реляционными базами данных через абстракцию ORM (SQLAlchemy), проектирование связей «один ко многим» между таблицами

* Разработка API: Создание RESTful API с использованием FastAPI, реализация CRUD операций и сложных аналитических запросов

* Микросервисная логика: Настройка взаимодействия между изолированными контейнерами через сеть Docker

* Оркестрация: Управление инфраструктурой приложения через Docker Compose

* Frontend разработка: Создание адаптивного пользовательского интерфейса с поддержкой тёмной/светлой темы и динамической загрузкой данных
